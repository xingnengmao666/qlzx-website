🛠️ 维护模式管理控制台

管理网站的维护状态 - 开启后，除主页外所有页面将显示 404

🔐 管理员验证

💡 提示：

这是您在部署时设置的 ADMIN_TOKEN 环境变量。

该密钥仅存储在本地浏览器，不会上传到服务器。

📊 当前状态
维护模式：检查中...
⚙️ 操作控制

⚠️ 说明：

开启维护模式：除主页外，访问任何子页面（/news.html、/email-apply.html 等）都会显示 404 页面。

关闭维护模式：网站恢复正常访问。

主页不受影响：无论维护模式开启或关闭，主页 (/) 始终可访问。

📚 API 使用说明

开启维护模式：

fetch('https://your-worker.workers.dev/api/maintenance/enable', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_ADMIN_TOKEN' } }) .then(r => r.json()) .then(data => console.log(data));

关闭维护模式：

fetch('https://your-worker.workers.dev/api/maintenance/disable', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_ADMIN_TOKEN' } }) .then(r => r.json()) .then(data => console.log(data));

查询状态：

fetch('https://your-worker.workers.dev/api/maintenance/status', { headers: { 'Authorization': 'Bearer YOUR_ADMIN_TOKEN' } }) .then(r => r.json()) .then(data => console.log(data));
